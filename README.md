# Zmail Clone: React + TypeScript + Vite

## Tech stack

- React
- Tailwind
- Shadcn

## Init project

```sh
npm create vite@latest .
```

## Setup tailwind

https://tailwindcss.com/docs/installation/using-vite

```sh
npm install tailwindcss @tailwindcss/vite
```

Modify vite.config.ts

## Setup of TS

```sh
 npm install -D @types/node
```
